# Multi API Business Intelligence Platform

## Business Problem
Companies use multiple systems and need one dashboard.

This project collects:
- Weather Data
- Currency Data
- Movie Trends
- News
- Country Statistics

and builds:
- ETL Pipelines
- SQL Database
- Analytics Layer
- Executive Dashboard
- Automated Reports

## Tech Stack
- Python
- SQLite
- SQLAlchemy
- Streamlit
- Pandas
- Requests
- Matplotlib
- Pytest
- GitHub
